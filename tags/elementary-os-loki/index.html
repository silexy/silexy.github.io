Voorzichtig afscheid van Windows

Nu mijn computer een beetje bejaard wordt, denk ik erover om het systeem een upgrade te laten ondergaan.  Daarbij zie ik meteen een nieuwe harde schijf in 't verschiet, die een SSD (solid state drive) zou moeten worden, omdat het lezen en schrijven nu de meeste tijd in beslag neemt. Het RAM geheugen zou ook wel een extra hulpchipje kunnen gebruiken. Erg kostbaar. Een nieuwe computer dan?

Eigenlijk is het onzinnig. Een tijdje geleden was deze computer toch prima? Met de nieuwe Windows 10 is het bergafwaarts gegaan, dat kan het oudje niet meer dragen.

Nu heb ik de pech dat ik een eerlijk betalende burger ben, dus ik heb een aantal software programma's waarvoor ik licentie heb betaald, en zeker de helft daarvan draait alleen op Windows. Geld uitgeven aan nieuwe hardware is jammer, maar geld weggooien omdat je een programma niet meer kan gebruiken is dat ook.

Toch wil ik voorzichtig proberen om Windows te verlaten. Vooral ook omdat het mijn stroom en computer gebruikt om allerlei privé zaken het internet op te sturen.
